# Angular Basics
A basic setup for learning and playing with AngularJS

## Primary Focus Libraries
- Angular JS
- UI-Router
- Twitter Bootstrap (3.x)
- Font-Awesome

## Included In This Package
- Angular JS (1.x)
- ngAnimate
- ngCookies
- ngResource
- ngTouch
- ngCookies
- ui.router
- Twitter-Bootstrap (3.x)
- Font-Awesome

### Getting Started

Install `npm` and `bower` dependencies

```JavaScript
$ npm install
$ bower install
```

Start browserSync

```JavaScript
gulp server
```
